# MyFirstApp
A simple and clean iOS Counter App built with SwiftUI.
This project demonstrates basic SwiftUI concepts such as State management, layout stacks, button actions, and UI styling.

📱 App Preview

A minimal counter application where users can:

Increase the counter

Decrease the counter

Reset the counter

🚀 Features

➕ Increase counter

➖ Decrease counter

🔄 Reset counter

🎨 Modern SwiftUI UI

✨ Spring animation when the value changes

🧊 Glass style buttons

📱 Works on all iPhone screen sizes

🛠 Built With

Swift

SwiftUI

Xcode


🎯 Learning Purpose

This project is useful for beginners learning:

SwiftUI layout (VStack, HStack, ZStack)

State management

Button actions

UI styling

SwiftUI animations
